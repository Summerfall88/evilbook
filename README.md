# Evilbook

Книжный блог Кристины: честные рецензии на книги, тёмные истории и художественное вдохновение.

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)
- Lucide React (Icons)
- Shadcn UI (Components)

## Разработка

1. Клонируйте репозиторий.
2. Установите зависимости: `npm install`
3. Запустите проект: `npm run dev`
4. Сборка проекта: `npm run build`

## Деплой

Сайт автоматически развертывается на GitHub Pages при пуше в ветку `main`.
Ссылка на проект: https://summerfall88.github.io/evilbook/
